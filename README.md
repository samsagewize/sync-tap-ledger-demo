# Sync Demo

Sync is an invited registration room for musicians who need a cleaner way to prepare song registrations and split paperwork. Phones join a room, install the app, answer whether they already have a PRO, complete rights profiles or start a guided PRO setup path, confirm percentages, and leave with a transparent registration packet.

The demo is also an installable Progressive Web App. It includes a web manifest, app icons, standalone display mode, and a service worker for offline shell caching.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project is Vercel-ready with `vercel.json`.
